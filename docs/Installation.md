# Installation

**Not yet published on Packagist or [PIE](https://github.com/php/pie).** This extension is still in development (see the
Classes section of the main `README.md` for what's implemented) and hasn't been tagged for release yet. The pages below
describe the intended end-user workflow once it is published, plus how to build and install it locally today from a
checkout of this repository.

Pick the page matching your situation:

- **[Mac, Linux](Installation/Mac,%20Linux.md)** - via [PIE](https://github.com/php/pie), or manually.
- **[Windows](Installation/Windows.md)** - no working install path yet; explains why, and how to build it yourself in
  the meantime.
- **[Docker](Installation/Docker.md)** - a `Dockerfile` build stage.
- **[Remote](Installation/Remote.md)** - VPS/cloud servers (DigitalOcean, EC2, etc.), including the production-server
  specifics (`phpenmod`, restarting `php-fpm`/Apache) that don't apply to a local dev install.
